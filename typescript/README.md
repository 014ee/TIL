# TypeScript

```
npm init -y // npm 세팅 (package.json 파일 생성)
```

```
npm i typescript -D // 타입스크립트 설치
npx tsc --init // 타입스크립트 환경설정 파일 생성 (tsconfig.json) 
npm tsc // tsconfig 설정에 맞춰 컴파일 실행 (문법적 에러에 대해서는 경고만 해주고 js 파일 생성)
```
